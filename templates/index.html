01 / CONCEPT

What is Naive Bayes?

01

Definition

Naive Bayes is a probabilistic classifier based on Bayes' theorem. It assumes all features are conditionally independent given the class — the "naive" assumption. Despite this simplicity, it performs well on text classification, spam filtering, and more.

02

Bayes' Theorem

P(C|X) = P(X|C) × P(C) / P(X)

  • P(C|X) — Posterior probability of class C given features X
  • P(X|C) — Likelihood: probability of X given class C
  • P(C) — Prior probability of class C
  • P(X) — Evidence (constant, often ignored)
03

Naive Assumption

Features are assumed independent:

P(X|C) = P(x₁|C) × P(x₂|C) × ... × P(xₙ|C)

So: Prediction = argmax P(C) × Π P(xᵢ|C)

04

Gaussian Naive Bayes

For continuous features, assume Gaussian distribution:

P(xᵢ|C) = (1/√2πσ²) × e^(-(xᵢ−μ)²/2σ²)

Mean (μ) and variance (σ²) are estimated from training data per class.

WORKED EXAMPLE

Play Tennis Dataset

# Outlook Temperature Humidity Wind Play?
1 Sunny Hot High Weak No
2 Sunny Hot High Strong No
3 Overcast Hot High Weak Yes
4 Rainy Mild High Weak Yes
5 Rainy Cool Normal Weak Yes
6 Rainy Cool Normal Strong No
7 Overcast Cool Normal Strong Yes
8 Sunny Mild High Weak No
9 Sunny Cool Normal Weak Yes
10 Rainy Mild Normal Weak Yes
11 Sunny Mild Normal Strong Yes
12 Overcast Mild High Strong Yes
13 Overcast Hot Normal Weak Yes
14 Rainy Mild High Strong No
S1
Prior Probabilities
P(Yes) = 9/14
64%
P(No) = 5/14
36%
S2
Interactive Predictor

P(Yes|...) ∝ P(Yes) × P(Sunny|Yes) × P(Hot|Yes) × P(High|Yes) × P(Weak|Yes)
= 9/14 × 2/9 × 2/9 × 3/9 × 6/9 = 0.0071

P(No|...) ∝ P(No) × P(Sunny|No) × P(Hot|No) × P(High|No) × P(Weak|No)
= 5/14 × 3/5 × 2/5 × 4/5 × 2/5 = 0.0274

→
Final Prediction

Since 0.0274 > 0.0071 → Play = No

G

Gaussian NB

Continuous features. Assumes normal distribution.

M

Multinomial NB

Discrete counts. Common in text classification.

B

Bernoulli NB

Binary features (yes/no, 0/1).

C

Complement NB

Variant for imbalanced datasets.

05 / HELP

How to Use This Lab

1

Prepare Your Dataset

Your CSV file must follow this format:

Feature1, Feature2, ..., Label Sunny, Hot, High, Yes Rainy, Cool, Normal, No
  • Last column must be the target label
  • All other columns are features
  • Both numeric and text values supported
  • First row must be column headers
2

Upload or Use Sample

Go to the Lab section and either:

  • Click Upload Dataset to use your own CSV
  • Click Download Sample Dataset for a ready-made example
  • Click Try Demo to auto-run without uploading
3

Run the Model

Click Run Naive Bayes to:

  • Train a Gaussian Naive Bayes model
  • See step-by-step probability calculations
  • View accuracy, confusion matrix, prediction table
  • Inspect per-sample class probabilities
4

Understand the Output

  • Accuracy — % of correct predictions
  • Prior — class frequency in training data
  • Likelihood — Gaussian parameters per class/feature
  • Confusion Matrix — actual vs predicted breakdown
  • Green rows — correct predictions; Red rows — errors
